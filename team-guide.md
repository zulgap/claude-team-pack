# 줄갭 작업 안내 (세션 시작 시 자동 주입 · 원격 업데이트)

> 이 안내는 매 세션 시작 시 GitHub에서 자동으로 최신본을 받아옵니다. (직원 PC 재설치 불필요)

## 쓸 수 있는 명령
- **`/시작`** 또는 "오늘 뭐부터" → 노션 마스터 허브에서 작업 현황을 불러와 무엇부터 할지 정리
- **`/저장`** 또는 "마무리해줘" → 오늘 한 작업을 노션 팀 세션저널에 5섹션으로 기록
- **`/블로그`** 또는 "블로그 써줘" → 주제만 주면 줄갭 사이트 형식대로 글을 작성·발행

## 쓸 수 있는 도구
- **노션**: 자료·정본 불러오기
- **PPT(pptx)**: 발표자료 생성
- **한글(hwp)**: 한글 문서 생성 (한컴오피스 필요)
- **제디(jedi)**: 회사 데이터(판단·도구) 조회 — 개인 토큰(JEDI_TOKEN) 등록 시에만 활성. 본인 권한·테넌트 안에서만, 읽기 위주. 토큰 없으면 이 도구만 안 보일 뿐 나머지는 정상.
