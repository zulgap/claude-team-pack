# 프리셋: 엔노블 (재혼·결정사)

> `/썸네일 엔노블 <대본>` 으로 호출 시 이 프리셋을 코어 알고리즘 위에 얹는다.
> 출처 정본: Notion 「유튜브 썸네일 가이드라인」(380efa2f) + 「엔노블 유튜브 가이드」(383efa2f).

## 채널 성격
- **타깃**: 재혼·결정사(결혼정보회사)를 고민/경험한 30~50대. "결혼", "재혼", "결정사" 관심층.
- **톤**: 따뜻·차분한 멘토. 불안·긴급 조장 금지, 안심되게.

## 🔴 배경 인물 = 반드시 한국인 (사장님 mandate 2026-07-13)
- 배경 생성(`ext_generate_image`) 프롬프트에 **항상 "한국인"을 명시 주입**한다.
- 표준 문구: `"30대 초반 한국인 전문직 여성/남성(자연스러운 한국인 외모, 검은 머리)"` 형태.
- 서양인·혼혈 외모 금지. 인물 사진(선택)을 받으면 그 인물 유지(합성).

## 갭언어 유형 우선순위 (STEP 2)
- 대본이 사연·심리형이면: **공감형 > 통념깨기 > 사회적증거** 순으로 3안 구성.
- 예: 공감형 "~한 분만 보세요" / 통념깨기 "결정사 해봤는데도…" / 사회적증거 "나만 몰랐던…".

## 배경 무드 (STEP 2 배경컨셉)
- 기본값 = **상담실 / 대화 / 노트** "실제 상담" 무드. 부드러운 자연광, 감성·신뢰.
- "증거"보다 **분위기로 신뢰**. 인물은 화면 한쪽 + 반대쪽 여백(텍스트 공간).

## 텍스트 스타일 (검증된 조합)
- 제목 = 화이트 Black + 검은 외곽선 + **빨강 강조**(핵심어 또는 아랫줄 전체).
- compose 인자: **`--acc red --stroke 4px`** (엔노블은 얇은 외곽선 + 빨강 강조).
- 서브카피 = 노랑 하단 자막.
- 기본 레이아웃 = **A**(하단 2줄) 또는 **B**(1인 인터뷰). 쇼츠면 C.
- 비율 = **16:9**(A/B) / **9:16**(C 쇼츠).

## 브랜드
- 우상단 워터마크 `엔노블`.

## 금칙 (민감정보)
- 실명 / 서류 캡처 / 회사 내부 / 구체 사연 과다 노출 **금지**.
- 결론 스포 금지("정답은 OOO") — 궁금증만 남긴다.

## 확정 예시 (정본)
