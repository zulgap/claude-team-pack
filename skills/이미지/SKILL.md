# 이미지 — 텍스트로 이미지 만들기 (팀원용)

원하는 그림을 말하면 줄갭 백엔드가 **GPT Image 2**(실패 시 Gemini 자동 폴백)로 이미지를 만들어 준다. 키 입력·설치 불필요 — 네 제디 토큰으로 회사 작업공간에 자동 저장된다.

> 팀원용: 프롬프트는 **한국어로 그대로** 말해도 된다(백엔드가 알아서 영문 변환). 너는 "무엇을, 어떤 비율로"만 정하면 된다.
> **제디(jedi) 도구가 활성일 때만 동작** — 개인 `JEDI_TOKEN` 등록 필요(없으면 `/시작`의 도구 안내 참고).

## 실행 순서

1. 사용자 발화에서 **무엇을 그릴지(프롬프트)** + **비율**을 파악한다. 비율은 픽셀이 아니라 종류로:
   - **정사각** `1:1` — 프로필·아이콘·SNS 정방형
   - **세로** `9:16`(풀세로) / `3:4`(살짝 세로) — 책표지·포스터·세로 카드·스토리
   - **가로** `16:9`(와이드) / `4:3`(살짝 가로) — 배너·썸네일·발표 슬라이드
   - 모르겠으면 정사각(`1:1`).
2. **생성 직전 1줄 미리보기**로 확인: `컨셉 / 비율 / 장수`. (사용자가 이미 "만들어"라고 명확히 했으면 바로 진행)
3. **`ext_generate_image` 도구 호출** (제디 MCP 브릿지 경유):
   - `prompt`: 그릴 내용 (**한국어 그대로** — 백엔드가 영문 자동 번역)
   - `aspect_ratio`: 위에서 정한 비율 (`1:1` / `9:16` / `3:4` / `16:9` / `4:3`)
   - `n`: 장수 (기본 1, 최대 8)
   - 참조 이미지가 **웹 URL**로 있으면: `mode: "edit"`(부분 수정) 또는 `"style_ref"`(스타일만 차용) + `input_images: [{ url, role }]` (`role` = `edit_target` / `reference`). 얼굴·스타일을 유지해야 하면 `preserve_fidelity: true`.
   - **회사(tenant)는 네 토큰으로 자동 결정** — `tenant_id` 등 회사 식별 인자는 **절대 넘기지 말 것**(서버가 강제).
4. 결과를 보고: 반환된 **`image_url`(이미지 링크)** 를 사용자에게 전달.
   - 정상: `✅ 이미지 생성 완료 — {비율}, {장수}장\n{image_url}`
   - 여러 장이면 링크를 모두 전달.

## 참조 이미지로 얼굴·스타일 유지

`input_images`는 **웹 URL**만 받는다. 유지하려는 사진이 인터넷에 올라간 URL이면 `mode: "edit"` + `preserve_fidelity: true`로 그 얼굴/스타일을 살린 변형을 만들 수 있다.
- 로컬 PC 사진(URL 없음)으로 얼굴 유지가 필요하면 v1에서는 아직 지원 안 함 — 사장님께 요청.

## 안 되는 경우

- "제디 도구가 안 보임 / `ext_generate_image` 없음" → `JEDI_TOKEN` 미등록. install 재실행 또는 사장님께 토큰 요청.
- **한글 텍스트는 모델이 자주 깨뜨린다** → 1차 결과는 *배경·구도·색감 시안*으로 보고, 최종 제목·문구는 디자인에서 재식자 권장.
- 유료 API 호출(비용 발생) — 요청이 명확할 때 진행.
