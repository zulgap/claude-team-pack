# 1Password 공용 금고 설정 (지금은 불필요 — 나중에 참고)

> ⏸️ **지금은 설정할 필요 없습니다.** 현재 직원 도구(노션·PPT·한글)는 공유할 비밀번호가 없습니다 (노션은 각자 로그인). 나중에 직원과 **유료 API 키**(예: 이미지·영상 생성)를 공유하게 되면 그때 이 가이드를 따르세요.

키를 카톡·파일로 주고받지 않고, **공용 금고**에 넣어 직원을 초대하는 방식입니다.

## 사장님 (1회 설정)
1. **1Password 가입** (가족 요금제 5명, 월 약 5~7천원)
2. **"줄갭 공용" 볼트(금고) 생성**
3. **공유할 키만 넣기** (콘텐츠용):
   - 노션 관련 키 등
   - ⚠️ **production 키(Railway·Supabase·Cloudflare)는 넣지 말 것** → 사장님 개인 볼트에만
4. **직원 2명 이메일 초대** → "줄갭 공용" 볼트 공유

## 직원 (1회)
1. 1Password 설치 + **초대 수락**
2. 끝. 키를 직접 보거나 복사할 일이 없습니다.

## Claude Code 연결 (선택 — 고급)
- `op run -- claude` 로 실행하면 금고 키가 실행 순간에만 자동 주입 (파일에 안 남음)
- 단, 노션은 `claude` 안에서 OAuth 로그인하면 키 없이도 됩니다 → 대부분 직원은 이걸로 충분

## ⚠️ 노션 권한도 꼭 주세요 (별개)
직원이 `/시작`·`/세션저널`을 쓰려면 노션 페이지 공유가 필요합니다:
1. **마스터 허브**(376efa2f...) → 직원 이메일에 공유
2. **팀 세션 저널 DB** → 직원 이메일에 공유

## 참고 — 더 저렴하게
- 1Password 대신 **Bitwarden**(거의 무료)도 동일한 역할을 합니다. 작은 팀엔 이걸로도 충분합니다.
